### view text {#view-text}

文本查看命令

[root@localhost ~]# cat

[root@localhost ~]# less    ##向前翻页(使用b),向后翻页(使用f),或者方向键

[root@localhost ~]# more  ##只能用回车符向下翻页

[root@localhost ~]# head    -n  数字  文件名       ##查看文件起始n行

[root@localhost ~]# tail        -n  数字 文件名       ##查看文件末尾n行

[root@localhost ~]# file  #查看文件数据类型
