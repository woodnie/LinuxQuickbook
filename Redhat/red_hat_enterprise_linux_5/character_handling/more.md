### more {#more}

more

NAME

    more - file perusal filter for crt viewing

空白鍵 (space)                :代表向下翻一頁

Enter                        :代表向下翻『一行』

/字串                        :代表在這個顯示的內容當中，向下搜尋『字串』這個關鍵字

:f                        :立刻顯示出檔名以及目前顯示的行數

q                        :代表立刻離開 more ，不再顯示該檔案內容

b 或 [ctrl]-b        :代表往回翻頁，不過這動作只對檔案有用，對管線無用。
